## 🛡️ 安全评估标准

### **安全安装原则**
1. **来源可信** - 来自知名开发者或官方
2. **许可证友好** - MIT-0或类似开源许可证
3. **更新活跃** - 最近有更新维护
4. **功能明确** - 功能描述清晰，无模糊表述
5. **依赖透明** - 依赖关系明确，无隐藏风险

### **风险评估**
- **高风险**: 需要系统权限、网络访问、文件操作
- **中风险**: 需要特定API访问、数据收集
- **低风险**: 只读操作、本地数据处理

## 🔧 安装前安全检查

### **系统状态检查**
1. **磁盘空间**: `df -h` (确保有足够空间)
2. **内存使用**: `free -h` (确保内存充足)
3. **网络连接**: `ping -c 3 clawhub.com` (确保网络正常)
4. **权限检查**: 确保有安装权限

### **技能验证步骤**
